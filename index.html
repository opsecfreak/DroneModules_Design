Project Overview

This interactive guide provides a complete roadmap for building the drone's Air Unit. The goal is to create a reliable, long-range electronics module capable of receiving commands from a Ground Unit to control a payload mechanism, like a servo-actuated drop system.

This application is designed to replace a static document with a hands-on, interactive experience. You can navigate through the required parts, follow the development milestones, and refer to the detailed assembly and troubleshooting guides as you build.

Required Skills

  • Basic electronics knowledge (voltage, ground, simple circuits).
  • Basic soldering skills. It's highly recommended to practice first if you're new.
  • Familiarity with the Arduino IDE for programming the STM32 microcontroller.

Development Roadmap

Follow these milestones in order to ensure a smooth development process. Check off each major milestone as you complete it to track your progress.

Objective: Purchase all components and verify each one works individually before soldering.

  • Order all parts from the BOM.
  • On a solderless breadboard, test the STM32 by uploading a "Blink" sketch.
  • Connect the Buck Converter to a battery and adjust the output to exactly 5V.
  • Wire the LoRa module and servo to the STM32 on the breadboard and run example code.

Objective: Create the permanent circuit on the perfboard.

  • Plan the physical layout of components for a compact fit.
  • Solder all modules and connectors onto the perfboard as detailed in the Assembly section.

Objective: Safely power on the fully assembled unit and check for critical errors.

  • Visually inspect all solder joints for bridges or cold joints.
  • Use a multimeter to check for short circuits between power and ground.
  • Connect the battery and verify the 5V and 3.3V power rails are correct.

Objective: Write and upload the final firmware to control the system.

  • Develop code to initialize the LoRa module and listen for incoming packets.
  • Develop code to control the servo based on commands received via LoRa.
  • Combine all code into a single, robust program.

Objective: Test the complete system with the Ground Unit and prepare for flight.

  • Perform range testing of the LoRa communication in an open area.
  • Verify the payload drop mechanism works reliably over multiple attempts.
  • Securely mount the finished unit to the drone.

Detailed Assembly Guide

This section details the physical construction of the circuit on perfboard. Follow these steps carefully. It is recommended to complete all power wiring before moving on to data wiring.

1Layout & Solder Components

Before soldering, arrange components on the perfboard for a compact layout. Solder pin headers to your modules first, then solder the modules and connectors securely to the perfboard.

2Power Wiring

Use a slightly thicker wire (e.g., 24 AWG) for these connections. Use RED for positive and BLACK for ground.

  • Solder the TVS Diode legs across the **Battery Connector** terminals.
  • Connect Battery Connector (+) to Buck Converter VIN+.
  • Connect Battery Connector (-) to Buck Converter VIN-. This is your System Ground.
  • Connect Buck Converter VOUT+ to the 5V pin on the STM32.
  • Connect Buck Converter VOUT- to any GND pin on the STM32.

3Data & Peripheral Wiring

Use thinner wire (e.g., 30 AWG) for these signal connections. Using different colors is highly recommended.

LoRa Module Wiring

  • VCC → STM32 3.3V
  • GND → STM32 GND
  • SCK → STM32 pin PA5
  • MISO → STM32 pin PA6
  • MOSI → STM32 pin PA7
  • NSS → STM32 pin PA4

Servo Connector Wiring

  • +5V Pin → STM32 5V pin
  • GND Pin → STM32 GND pin
  • PWM Pin → STM32 pin PA8

Troubleshooting Guide

If you encounter issues, consult this guide. Click on a problem to reveal the possible causes and solutions.

  1. Check that battery polarity is correct (+ to + and - to -).
  2. Use a multimeter in continuity mode to check for short circuits between the main power and ground wires.
  3. Verify the buck converter is adjusted and outputting 5V.

  2. Ensure the Ground and Air units are set to the same frequency and sync word in the code.
  3. Check that the antenna is securely connected to the LoRa module's u.FL connector.
  1. Verify the servo has a solid 5V and GND connection directly from the buck converter's output if possible.
  2. Check the PWM signal wire connection to pin PA8.
